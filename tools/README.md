# Caja de herramientas

Esta carpeta contiene utilidades para mantener y evolucionar el sitio.

## migration
Scripts utilizados durante la migración desde Blogger.

## audit
Herramientas para revisar enlaces, imágenes y contenido.

## images
Procesamiento y comprobación de imágenes.

## deploy
Automatizaciones relacionadas con la publicación y mantenimiento.
