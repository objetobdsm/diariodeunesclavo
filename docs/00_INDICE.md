# Documentación del proyecto

Esta carpeta reúne toda la documentación relacionada con el desarrollo, mantenimiento y evolución de **Diario de un esclavo**.

Su objetivo es facilitar la continuidad del proyecto, documentar las decisiones tomadas y servir como referencia para el trabajo cotidiano.
