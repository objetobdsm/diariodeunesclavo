### 15 de julio

Creación del Manual del editor.

Se documenta el proceso de publicación, mantenimiento y conservación del proyecto.

Se reorganiza definitivamente el proyecto.

- El diario pasa a su ubicación definitiva en `~/Escritorio/Descargas/Proyectos/diario`.
- Se recupera un repositorio Git limpio desde GitHub tras detectar una copia local dañada.
- Se verifica la coincidencia entre la versión local, GitHub y Netlify.
- A partir de esta fecha el proyecto continúa su desarrollo desde la nueva ubicación.
