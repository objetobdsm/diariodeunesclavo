# Diario de un Esclavo

> Archivo editorial construido con Hugo.

## ¿Qué es este proyecto?

**Diario de un Esclavo** reúne más de veinte años de escritos personales sobre BDSM, relaciones D/s y otras reflexiones.

Tras su migración desde Blogger, el proyecto evoluciona hacia una plataforma editorial propia, diseñada para crecer sin perder la sencillez, la legibilidad y el protagonismo del contenido.

El objetivo no es únicamente conservar el archivo histórico, sino convertirlo en un espacio vivo donde los lectores puedan recorrer el diario por diferentes caminos: cronológicamente, por temas o mediante futuras secciones como la Biblioteca o los Escritos.

## Filosofía del proyecto

El contenido es siempre el protagonista.

Toda decisión técnica, editorial o visual debe responder a una única pregunta:

> **¿Hace más agradable recorrer este lugar?**

## Objetivos

- Conservar el archivo completo.
- Facilitar la lectura.
- Mejorar la exploración temática.
- Crear una arquitectura preparada para crecer.
- Mantener un diseño sobrio, minimalista y atemporal.
